## Тест на логику

Тест поможет вам понять, присутствует ли у вас логика.
Написан на языке программирования Ruby.

### Для запуска вам нужно: 

* Необходима версия Ruby не ниже 2.x.x
* Для запуска наберите в командной строке:

```ruby
ruby logic_test.rb
```
